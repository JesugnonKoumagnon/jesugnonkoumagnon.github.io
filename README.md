## Structure

```
index.html              Homepage
research.html           Research & applied projects
talks.html              Talks & seminars
cv.html                 Full CV
papers/                 PDF reports, papers, and thesis
projects/               Project-specific code and notes pages
  ├── cervical-cancer-colposcopy/    notes.html
  ├── boko-haram-climate-lakechad/   notes.html
  ├── nigeria-mpi-social-protection/ notes.html
  ├── unicef-poverty-dashboard/      notes.html
  ├── firm-training-impact/          notes.html
  └── benin-saving-rct-hfc/          notes.html + scripts/
code/                   Standalone analysis code (R scripts, Jupyter notebook + HTML render)

```

Each folder under `projects/` has a styled `notes.html` page describing the methodology, data sources, and (where available) the underlying code.
